# 第 1 章 | 课程导学与准备工作
## 1-1 | 课程导学
### 学习的时长
- 视频时长约 38 小时
- 每天学习 6 小时，大概需要二至三个月
- 学习的关键是要动手写代码，运行测试它的效果
### 技术储备要求
- 掌握了 C/C++ 语言的基础知识，能编写 50-100 行的 demo 程序
- 熟悉 Linux 操作系统的疮痈命令和 vi 工具
